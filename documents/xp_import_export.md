# 경험치 내보내기/가져오기 표준
마늘봇의 경험치 내보내기/가져오기 파일은 json 파일을 사용하며 그 양식은 다음과 같습니다.

이 파일 양식이 표준이 되기를 원하며, 따라서 다른 봇에서 경험치 데이터 내보내기/가져오기 기능 개발 시 이 json 파일 형식을 그대로 사용하셔도 됩니다. 구체적인 사용법은 아래로 스크롤하시면 읽으실 수 있습니다.

```json
{
    "file_version": 1,
    "data": [
        {
            "user_id": 1305492487137267722,
            "xp": 9065
        },
        {
            "user_id": 1122876643652931674,
            "xp": 1000
        },
        {
            "user_id": 1273450447633645600,
            "xp": 2200
        }
    ]
}
```

 - `file_version`: 파일 표준 버전입니다. 현재 버전은 1입니다.
   - 파일 유효성 체크 등에 사용됩니다.
 - `data`: `user_id`와 `xp`를 한 쌍으로 묶어서 유저들의 경험치 값을 저장해두는 곳입니다.
